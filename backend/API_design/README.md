## Backend Layer's Information


## authentication.json file

This file contains mock API endpoints designed to integrate with Postman's interface for authentication testing.
Includes Signup, Signin, Forgot password and other authentication scenarios.
Each endpoint is enhanced with HTTP status codes and examples for better testing.

## swagger_editor.json file

Mock api json file documented and tested by Swagger(OPEN API).
Swagger Editor provides API requests responses details and authentication flows.

Folllow the instructions to use swagger editor:

1- Download the swagger_editor.json file.
2- Open Swagger Editor page, then click the File button.
3- Import the json file.
So, you can test authentication module.
